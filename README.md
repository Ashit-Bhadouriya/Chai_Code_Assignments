# 📄 Resume Page in HTML

A clean, static personal resume/portfolio page built with pure HTML. It presents professional information in a structured, readable format without any external dependencies.

---

## 📁 Project Structure

```
resume-page/
└── index.html       # Main (and only) HTML file containing the full resume
```

---

## 🚀 Getting Started

No installation or build tools required. Just open the file in any browser:

```bash
# Option 1 — Double-click
Open index.html in your file explorer

# Option 2 — Terminal
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux
```

---

## 📋 Sections

| Section | Description |
|---|---|
| **Header** | Name and professional titles |
| **About Me** | Short personal bio and area of expertise |
| **Skills** | List of technical skills |
| **Experience** | Work history in a tabular format |
| **Projects** | Featured projects with tech stack and timeline |
| **Education** | Academic qualifications in a tabular format |
| **Contact** | Email, phone, and location |

---

## 🛠️ Built With

- **HTML5** — Semantic structure and layout
- No CSS, JavaScript, or external libraries required

---

## ✏️ Customization

To adapt this resume for yourself, open `index.html` in any text editor and update:

- `<h1>` — Your full name
- `<p>` under the name — Your job titles
- All table rows under **Experience** and **Education**
- `<ul>` items under **Skills**
- Project descriptions and tech stacks
- Contact details (email, phone, location)

---

## 📌 Notes

- The experience and education sections use an HTML `<table>` with a `border="1"` attribute for basic visible borders. For a more polished look, consider adding a CSS stylesheet.
- The page is fully static — no backend or server required.
